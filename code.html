The Code for ReStringify


                        //Get the String from the page
                        function getString(){
                        
                            document.getElementById("alert").classList.add("invisible")
                        
                            let userString = document.getElementById("userString").value;
                        
                            let revString =  reverseString(userString);
                        
                            displayString(revString)
                        }
                        
                        //Reverse the String
                        function reverseString(userString){
                        
                            let revString = [];
                            for(let i = userString.length-1; i >= 0; i--){
                                revString += userString[i];
                            }
                        
                            return revString
                        }
                        
                        //Display the reversed string to the user
                        function displayString(reversedString){
                            
                            document.getElementById("msg").innerHTML = `Your String Reversed is: ${reversedString}`;
                            document.getElementById("alert").classList.remove("invisible");
                        
                        }

The Code is structured in three functions.

getString

getString is the controller function that calls reverseString and displayString using the user's inputted string. The function starts off by making the alert box invisible by adding "invisible" to its class list. "invisible" is a bootstrap class that makes an object invisible, which is what I want for the alert box until the button is clicked. Then the function gets the inputted string by grabbing it from the DOM using getElementById. getString then uses this string to call reverseString to reverse the string and display it using displayString.

reverseString

reverseString reverses the string given by the parameter. To do so the function creates a new string array and loops through the inputted string starting from the end. During each iteration, the current letter is added into the new string. This loop continues on until all letters are added in reverse into the new string. reverseString returns the reversed string of the inputted string"

displayString

displayString is a function that displays the reversed string to the user. The function does so by adding a template literal into the message's innerHTML and removing the "invisible" class from the alert, showing the message to the user.
